安装：需要一台有MySql和Tomcat和JDK的服务器

  1. 找到Blog压缩包中resource目录下的建表的sql语句,并按照要求创建数据库。
  2. 将Blog压缩包直接上传的Tomcat的webapp目录下。
  3. 在浏览器中访问博客地址,并到注册账号页面注册账号。
  4. 登陆的密码会发送到注册的邮箱中,请牢记生成的随机密码,登陆后及时修改原始密码。

最低系统需求

  • JDK 1.6或更高版本。
  • MySQL 5.0或更高版本。
  • Tomcat 6.0或更高版本。

系统推荐

  • JDK 1.8或更高版本。
  • MySQL 5.5或更高版本。
  • Tomcat 8.0或更高版本。
